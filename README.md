# Task Manager API

A simple REST API to manage tasks using Node.js, Experss.js, and MongoDB.

## Features
- Create read, update, delete tasks
- MongoDB Atlas for cloud DB
- RESTFUL architecture

## Routes
- GET `/api/tasks`
- POST `/api/tasks`
- GET `/api/tasks/:id`
- PUT `/api/tasks/:id`
- DELETE `/api/tasks/:id`

## Tech
- Node.js
- Express
- MongoDB (Mongoose)
